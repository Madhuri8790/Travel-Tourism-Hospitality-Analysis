# Travel, Tourism & Hospitality Analysis

## Project Overview

This project analyzes tourism and hospitality data to identify customer trends, seasonal demand, hotel performance, and revenue patterns.

## Objectives

- Clean and preprocess tourism data
- Perform Exploratory Data Analysis (EDA)
- Generate business insights
- Build an interactive Power BI dashboard
- Support data-driven decision-making in the tourism industry

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab
- Power BI
- Git & GitHub

## Dataset

Tourism & Hospitality Industry Analysis Dataset
